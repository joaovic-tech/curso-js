# Padrões de projetos com TypeScript

- As pastas estão organizadas com as categorias dos padrões de projetos segundo o livro da GoF. São elas, behavioural, creational e structural.
- Cada pasta contém um `README.md` detalhado a categoria em si e os padrões de projetos nela presentes.
- Cada padrão de projeto está dentro de uma pasta com seu nome. Existe um README descrevendo o padrão, um ou mais diagramas e trechos de código para exemplificação de implementação.
